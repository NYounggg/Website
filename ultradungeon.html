ABOUT

ULTRADUNGEON is a high-octane dungeon crawler currently in development using C# in the Godot game engine.


This is a project made by two friends to create a game that we enjoy. We hope that others will find it fun too.

CLASSES

ULTRADUNGEON has three main classes: Mage, Ranger and Melee, each with their own sets of gear. Players can mix and match weapons, armour and perks to create their own unique builds.

RELEASE DATE

ULTRADUNGEON has no planned release date as of now. We work on it in our spare time and hope to release it one day.


We are planning to release demo builds and progress updates as the game continues development.
